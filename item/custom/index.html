About

MCBE Essentials uses a custom-coded item rendering system with item data from TheDestruc7i0n and identifiers from bridge. You can force this website to recognize custom items by uploading them here.

Is this global?

Yes, it's global. This means that every app on this website will recognize your custom items.

What do I do?

Simply click "Upload" and select your PNG file that you use to display the item in-game. Type in the identifier of your item, and it should now display across the website as a custom item. Please limit your uploads to 8x8, 16x16, 32x32 or 64x64 PNGs to avoid issues in the website.
Note: once you've uploaded an asset, you can't edit it.
