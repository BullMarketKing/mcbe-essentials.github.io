Advanced Notation You can include math and variables in your code with . The following variables are available:
line - A number that starts at 0 and increases with each cycle.
length - The maximum amount of times that the code cycles (the length of the list, or, the amount defined in "Loop __ times").
value - ("Loop through a list" only) The current item in your list
Use in place of "value" if you want to subject the value to math.
