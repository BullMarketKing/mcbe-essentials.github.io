Loopr Reverse

Loopr Reverse is a simple app that will recognize repeated and static data, and separate them for you.

Input





Write in the text that gets repeated on each line here. Remember to use to let Loopr know where the changing value is in the line.

Output


Quick tip: Select one line in the output, and press Ctrl + A. It'll select the whole output for you. Then you can do Ctrl + C to copy it to your clipboard.
